# Kanban To-Do Board

A responsive, high-performance Kanban board application built entirely with vanilla web technologies. This application provides persistent, client-side task tracking using streamlined drag-and-drop mechanics.

Live Demo: [Insert Your Live GitHub Pages URL Here]

## 🚀 Key Features

* **Native Drag-and-Drop Interaction:** Built using the HTML5 Drag and Drop API for fluid task movement across columns.
* **Persistent Session Architecture:** Integrates client-side `localStorage` caching to guarantee zero data loss on browser refresh.
* **Modern UI Custom Layout:** Uses unified CSS custom properties (design tokens) and responsive Flexbox structures for adaptive styling.
* **Input State Validation:** Restricts invalid tasks and empty data forms using proactive DOM event checking.
* **Optimized Trash Zone:** Features an inline-SVG, floating action drop point for fast item deletion.

## 🛠️ Tech Stack & Methods

* **Structure:** Semantic HTML5 Markup
* **Styling:** CSS3 (Custom Variables, Flexbox Engine, Transitions)
* **Logic:** Vanilla JavaScript (ES6+, Event Delegation, Web Storage API)

## 📦 Local Installation & Setup

1. Clone this repository locally:
   ```bash
   git clone https://github.com
   ```
2. Navigate directly into the project folder:
   ```bash
   cd kanban-board
   ```
3. Open the main file in your preferred web browser:
   ```bash
   open index.html
   ```
